Device Tree for Lenovo P1m

Shipped Android Version | 5.1, 6.0
Storage | 16GB
MicroSD | Up to 64GB
Battery | Li-Pol 4000mAh battery
Display | 720 x 1280 pixels, 5.0 inches
Camera  | Main 8MP / Front 5MP, autofocus, LED flash
